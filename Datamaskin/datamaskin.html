Datamaskinens oppbygging og virkemåte

Prosessor

Prosessoren er hjernen i datamaskinen. Hastigheten på en CPU er mellom 2 - 4 Ghz for nye maskiner. CPU-en utfører instruksjonene gitt i et dataprogram, og den er det primære elementet som gjennomfører datamaskinens funkjsoner

Intel

Amerikansk produsent av prosessor. Grunnlagt 1968 - laget intel 4004 CPU, 4 bits cpu. Dagens CPUer er 64bit. 4 bit gir 2^4 som største tall = 16. For 5-6 år siden var standard størrelse 32 bit. Dagens CPU kan jobbe med tall på 2^64 = 18446744073709551616.

AMD

AMD (Advanced Micro Devices) laget sin første CPU 8080 AM9080 (en klone av intel 8080).

ARM

Acorn (Advanced) RISC (Reduced Intruction Set Computer) Machine. RISC CPUer har få og enkle instruksjoner (i motsetning til intel). Hver instruksjon skal ta minst mulig tid. Kompliserte beregninger består av mange instruksjoner. Brukes i mobiler (nesten alle?)

RAM

Ram er en forkortelse som betyr Random Access Memory

Lagring

Standard lagringsmedia idag er SSD. Tidligere (for få år siden) var HDD det mest brukte. SSD står for Solid state drive. siden den ikke har bevegelige deler, er søketiden kortere enn for mekanisk harddisk og den tåler støt bedre. HDD står for Hard drive disk.
